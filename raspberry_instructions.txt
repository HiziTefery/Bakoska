INSTALACIA PYTHONa
$ sudo apt-get install python-dev
INSTALACIA PIPu na dotahovanie packageov jazyka PYTHON
$ sudo apt-get install pip
INSTALACIA python-mysqldb API na pristup k databaze prostrednicvom jazyka PYTHON
$ sudo apt-get install python-mysqldb

protokolu MODBUS v jazyku PYTHON
$ sudo pip install -U pyserial
INSTALACIA samotnej kniznice minimalModbus
$ sudo pip install -U minimalmodbus
